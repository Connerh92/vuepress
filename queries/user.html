User Query

Notice

We have attached an extra organization filter criteria to the native User Query. You can utilize this param on any User Query

# Params

All of the standard User Query params are are available. In addition, the following params are also available:

PropertyTypeDescription
organizationstring, string[], integer, integer[], Organization, Organization[], nullThe organization criteria that the resulting users must have

# Usage

#setOrganization( $value )

Returns: User Query

ArgumentAcceptsDescription
$valuestring, string[], integer, integer[], Organization, Organization[], nullThe organization criteria that the resulting users must have

::: code

{% set query = craft.users.find({
    organization: {
        id: 1,
        organizationType: [1,2],
        userType: [1,2]
    }
}) %}

$query = User::find()->setOrganization([
    id => 1,
    organizationType => [1,2],
    userType => [1,2]
]);

:::
